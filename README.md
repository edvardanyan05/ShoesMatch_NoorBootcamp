# ShoesMaster

This team project is a casual Android puzzle game built during Noor Bootcamp by Noor Games. Match pairs of same-colored shoes and send them to the correct box before the container fills up. Fill the container with unmatched colors and it's Game Over!

## How to Play
- Tap a shoe to place it into the container
- Match two shoes of the same color
- They automatically fly into the correct box
- Fill the container with unmatched colors = Game Over

## Levels
- Level 0 — Tutorial (1 pair)
- Level 1 — 3 pairs
- Level 2 — 4 pairs
- Level 3 — 5 pairs
- Level 4 — 6 pairs
